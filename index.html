A SaaS Docker Management Dashboard

Features

Kraven comes with tons of well crafted features.

Manage Docker Environments

Now you have one place to manage all different docker environments or swarm clusters! It allows you to manage your docker containers, images, volumes, networks and much more!

Easy to Install and Manage

Auto Installer, Admin management page to control application global settings, Prometheus integration for metrics , docker compose file to run the Application, Workers, RabbitMQ, Redis, Prometheus and Grafana!

Asynchronous Everything

Everything is Asynchronous with ability to run any number of workers to scale. It can run as centralized or distributed application.

Well Secured

Two layers of encryption to secure your hosts.

Open Source Code

You can see the code of Kraven on GitHub. If you killed a bug, miss any feature feel free to create a pull request there, we'll definitely check it.
